## <Icon icon="pencil" /> Nicknames

The ability to edit the nickname of the user after they verified. This setting is **fully optional** and is not required.

<Tabs>
  <Tab title="Presets">
    We offer multiple different presets to choose from:

    **Display Only**: `{rbx_display}`

    **Username Only**: `{rbx_username}`

    **Display (@username)**: `{rbx_display} (@{rbx_username})`

    **Username (Display)**: `{rbx_username} ({rbx_display})`
  </Tab>
  <Tab title="Custom">
    We also offer the abillity to use a **custom nickname template**. You can view all the variables for this module in the [**Variable Reference**](/variables) and find the **Plugin Variables** section and click on **Verification**.
  </Tab>
</Tabs>

## <Icon icon="shield-quartered" /> Roles

We also allow the user to edit the roles that are given or removed after verification is completed.

**Verified Roles**: Verified roles are the roles that are given after the user has verified.

**Unverified Roles**: Unverified roles are the roles that are removed from the user after they verify.
